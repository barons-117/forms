Technology stack

Reliable infrastructure at every layer

Database & Auth

Supabase

Every form submission is written instantly to a PostgreSQL database with Row Level Security. Only authenticated admin users can read submissions — public users can only insert. Real-time and reliable.

Email Delivery

Brevo (Sendinblue)

Transactional email sent via Supabase Edge Functions through Brevo's SMTP API. Emails are delivered from your own domain (no-reply@tofes.pro) with DKIM authentication for maximum inbox deliverability.

Frontend

React + Vite

Built with React 18 and Vite for instant local development and optimized production builds. Deployed to GitHub Pages with a custom domain. Zero server costs, 100% uptime on CDN.

Tracking

Facebook Pixel

Per-form Pixel ID injection at runtime. PageView fires on load, Lead event fires on successful submission. Fully isolated — different forms can target different ad accounts.

Submission flow

What happens when someone hits submit

1

Validation

Client-side validation in the form's language. Errors shown as floating bubbles — no layout shift.

2

Database write

Submission saved to Supabase with timestamp. Field values stored as structured JSON, keyed by field ID.

3

Pixel event

Facebook Lead event fired if a Pixel ID is configured for this form.

4

Email dispatch

Edge Function calls Brevo API. HTML email with field-label table delivered to all configured recipients.

5

Thank you

User redirected to custom thank-you page or external URL, per form configuration.

Common questions

Frequently asked

Do I need to write any code to use tofes.pro?

No. The admin panel handles everything — form creation, field configuration, color customization, and publishing. Forms are live at a unique URL the moment you save.

Where is my data stored?

All submissions are stored in a PostgreSQL database hosted on Supabase in Frankfurt, Germany (EU West). Data is protected by Row Level Security — only authenticated admins can access it.

How quickly are email notifications sent?

Email notifications are triggered immediately after each submission via a serverless Edge Function. Typical delivery time is under 10 seconds from form submission to inbox.

Can I use the same form for Hebrew and English audiences?

Each form is set to a primary language (Hebrew or English), which controls the grid direction and error message language. You can mix field directions within a single form — for example, a Hebrew form with an English-direction email field.

Does the Facebook Pixel track across different forms?

Each form has its own Pixel ID setting. This lets you assign different ad accounts or campaigns to different forms — useful when running multiple clients or products from the same admin account.

Can I export my submissions?

Yes. The admin submissions panel includes a CSV export button that exports the currently filtered view. The export includes a UTF-8 BOM for correct Hebrew character rendering in Excel.
