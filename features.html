Core features

Built for results, not just aesthetics

Setup

Live in under 2 minutes

Create a form, add fields, choose colors, set recipients. No configuration maze, no developer required. Your URL is ready the moment you click publish.

Language

Field-level direction control

Mix Hebrew and English in the same form. Each field has its own RTL or LTR setting, fully independent of the overall form direction.

Alerts

Instant email alerts

A formatted email hits every recipient the moment someone submits. No polling, no delays, no missed leads.

Data

Submission dashboard

Search, filter by date, sort by any column, export to CSV. Every response is stored and searchable.

Tracking

Per-form Facebook Pixel

Assign a unique Pixel ID to each form. The Lead event fires on submission — real data, per campaign, per client.

Design control

Your form, your brand

Colors

Nine color controls

Background, card, title, labels, borders, input text, button, accent bar — each form is a blank canvas.

Layout

12-column layout grid — up to 6 fields per row

Place fields side by side with granular width control. Set each field's column span independently — half, third, sixth — and drop the submit button anywhere in the grid. The layout collapses cleanly to single column on mobile with zero configuration.

Width presets

Four form width presets

520 · 700 · 900 · 1100px. Always full-width on mobile regardless of selection.

Compliance

GDPR consent checkbox

Consent with hyperlinked terms, required enforcement, and a live preview while editing.

Thank you

Custom thank-you experience

Branded message, custom button with label and URL, or a direct redirect — per form.
