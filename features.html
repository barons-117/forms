Core features

Built for results, not just aesthetics

Setup
⚡

Live in under 2 minutes

Create a form, add your fields, choose colors, set recipients. No configuration maze, no developer required. Your form URL is ready the moment you click publish. Share it anywhere — ads, bio links, email campaigns, landing pages.

Language
🌍

Field-level direction control

Mix Hebrew and English in the same form. Each field has its own RTL or LTR setting, independent of the form's overall language.

📬

Instant email alerts

A formatted email lands in every recipient's inbox the moment someone submits. No polling, no delays, no missed leads.

📊

Submission dashboard

Search, filter by date, sort by any column, and export to CSV. Every response is stored and searchable.

🎯

Facebook Pixel per form

Add a unique Pixel ID to each form independently. Track leads and optimize ad campaigns with real conversion data.

Design control

Your form, your brand

🎨

9 color controls per form

Background, card, title, labels, input borders, input text, button background, button text, and accent bar — each independently adjustable.

📐

12-column layout grid

Place fields side by side with up to 6 columns per row. Set each field's width — half, third, quarter, or full — and position the submit button anywhere in the grid.

📏

4 form width presets

Choose from 520px, 700px, 900px, or 1100px for the form card width. Always full-width on mobile regardless of selection.

✅

GDPR consent checkbox

Add a consent checkbox below the submit button with custom text, hyperlinks, and required enforcement. Includes preview while editing.

💬

Custom thank-you experience

Set a personalized thank-you message, a custom button with your own label and URL, or a direct redirect to any destination.

🔤

Custom submit button text

Change the submit button label to anything: "Register", "Send", "Join us", "הירשם" — full control over the call to action.

Field types

Every field type you need

Text

Free-text input for names, addresses, and any short answer. Configurable direction and placeholder alignment.

Email

Email input with built-in format validation. Always left-aligned regardless of form direction.

Phone

Phone number field with format validation. Accepts international formats.

Dropdown (Select)

Comma-separated list of options rendered as a clean dropdown. Great for segmenting your leads.

Checkbox

Single checkbox for quick yes/no answers. Position adapts to form language — checkbox left for English, right for Hebrew.

Long text (Textarea)

Resizable text area for messages, comments, or longer responses. Respects RTL/LTR direction settings.
